# rondavizinho-site

Landing page do **RondaVizinho** — o vigia inteligente da sua rua.

Página estática (HTML + CSS, sem build), publicada via GitHub Pages.
Identidade visual: logo em SVG (`assets/`), paleta azul-confiança + verde,
tipografia Poppins/Inter.

## Rodar localmente

```bash
python3 -m http.server 8000
# abra http://localhost:8000
```

Projeto comunitário, sem fins lucrativos. Licença Apache 2.0.
